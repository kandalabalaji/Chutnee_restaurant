1. *Clone the repository*
   \\\`bash

   cd chutnee-restaurant
   \\\`

2. *Install dependencies*
   \\\`bash

   # or

   \\\`

   \\\`bash

   \\\`

4. *Open your browser*
   Navigate to [http://localhost:3000](http://localhost:3000)
